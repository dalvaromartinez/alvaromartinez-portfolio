League of Legends Outcome Predictor (UNIR)

Predictive model to estimate match outcomes using engineered features from match statistics. Trained and evaluated in Amazon SageMaker with scikit-learn/XGBoost.

What I built

  • EDA on match history (objectives, economy, damage, map control).
  • Feature engineering (gold diff per minute, vision control, objective pacing).
  • Modeling with XGBoost / Logistic Regression; cross-validation + ROC-AUC.
  • Explainability with feature importance / SHAP-style analysis.

Tech stack

Python pandas scikit-learn XGBoost SageMaker
